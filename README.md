# CyberCraft Website

CyberCraft Minecraft server veb-sayti — Next.js bilan qurilgan.

## Texnologiyalar

- Next.js (TypeScript)
- Tailwind CSS
- shadcn/ui komponentlar

## O'rnatish

```bash
# Paketlarni o'rnatish
npm install

# Development serverni ishga tushirish
npm run dev
```

Sayt http://localhost:3000 da ochiladi.

## Tuzilma

- `app/` — Next.js sahifalar va routing
- `components/` — Qayta foydalaniladigan komponentlar
- `hooks/` — Custom React hooks
- `lib/` — API client va utility funksiyalar
